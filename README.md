### commit 描述

* `fix` : 修改bug
* `feature` : 新功能
* `doc` : 文档改变
* `style` : 代码格式
* `refactor` : 对已有功能重构
* `perf` : 性能优化
* `test` : 增加测试
* `db` : 数据库脚本
* `comments` : 注释
* `version` : 版本号修改
